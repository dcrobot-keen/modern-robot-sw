# Wikilinks 사용법

Wikilinks는 Obsidian과 Quartz에서 다른 노트를 연결하는 방법입니다.

## 기본 문법

다른 노트로 링크하려면 `[[노트 이름]]` 형식을 사용합니다:
- [[index|홈으로 돌아가기]]

## 장점

- 빠르고 직관적인 링크 생성
- 자동 완성 지원 (Obsidian에서)
- 양방향 링크 자동 생성
- 파일명 변경 시 자동 업데이트

## 예제

일반 마크다운 링크: [GitHub](https://github.com)
Wikilink: [[index]]

Quartz는 두 가지 형식을 모두 지원합니다!
